Cryptography Ciphers in Python

This repository showcases Python implementations of several classical cryptographic ciphers. These ciphers serve as a foundation for understanding the evolution of encryption techniques. Below is the list of implemented encryption methods:

Included Ciphers

Caesar Cipher: A substitution cipher that shifts each letter by a set number of positions in the alphabet.

Monoalphabetic Cipher: A substitution technique where each plaintext letter is mapped to a corresponding letter in a shuffled alphabet.

Playfair Cipher: A digraph-based encryption method utilizing a 5x5 matrix for encoding letter pairs.

Hill Cipher: A polygraphic encryption system that relies on matrix multiplication for encoding.

Prerequisites

To execute these programs, ensure Python is installed on your machine. Verify the installation by running:

python --version

If Python is not installed, download it from python.org and complete the installation.

Running the Ciphers

Each cipher is implemented as an independent Python script. Follow these steps to run them:

Clone the repository:

git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name

Execute a specific cipher script:

python caesar_cipher.py
python monoalphabetic_cipher.py
python playfair_cipher.py
python hill_cipher.py

Follow the prompts to encrypt or decrypt messages using the chosen cipher.
