🔐 Cryptography Ciphers in Python

This repository provides Python implementations of fundamental classical cryptographic ciphers. These encryption techniques illustrate the evolution of cryptography and its foundational principles. Below are the ciphers included in this project:

🔢 Implemented Ciphers

Caesar Cipher 🏛️ - A basic substitution cipher that shifts letters by a fixed number of positions in the alphabet.

Monoalphabetic Cipher 🔄 - A substitution method where each letter in the plaintext is replaced with a corresponding letter from a predetermined shuffled alphabet.

Playfair Cipher 🔠 - A digraph-based encryption method that encodes pairs of letters using a structured 5x5 matrix.

Hill Cipher 📊 - A polygraphic cipher leveraging matrix multiplication for encryption.

⚙️ Prerequisites

Ensure Python is installed on your system before running the ciphers. You can verify your installation with:

python --version

If Python is not installed, download it from python.org and follow the installation instructions.

🚀 Running the Programs

Each cipher is a standalone script. To execute them, follow these steps:

1️⃣ Clone the Repository

git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name

2️⃣ Run a Specific Cipher

python caesar_cipher.py
python monoalphabetic_cipher.py
python playfair_cipher.py
python hill_cipher.py

📌 Follow the on-screen instructions to encrypt or decrypt messages using the chosen cipher.
